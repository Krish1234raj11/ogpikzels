Getting Started

Welcome to Pikzels Network! Here's everything you need to know to get started:

Connecting to the Server

Server IP: play.pikzels.net

First Steps

  1. 1. Read the rules with /rules
  2. 2. Claim your starter kit with /kit starter
  3. 3. Find a place to build with /rtp
  4. 4. Join our Discord server for updates and community

Basic Commands

/spawn - Teleport to spawn

/home - Teleport to your home

/sethome - Set your home location

/tpa [player] - Request teleport to player

/bal - Check your balance

Commands

Complete list of available commands on Pikzels Network:

General Commands

/help - Show help menu

/rules - Display server rules

/info - Server information

/list - Show online players

/who [player] - Player information

Teleportation Commands

/spawn - Teleport to spawn

/home [name] - Teleport to home

/sethome [name] - Set home location

/delhome [name] - Delete home

/tpa [player] - Request teleport

/tpaccept - Accept teleport request

/tpdeny - Deny teleport request

/rtp - Random teleport

Economy Commands

/bal - Check balance

/pay [player] [amount] - Pay player

/shop - Open shop menu

/sell hand - Sell item in hand

/sell all - Sell all items

Frequently Asked Questions

Common questions and answers:

Q: How do I claim land?

A: Use a golden shovel to select corners, then type /claimland to protect your area.

Q: Can I have multiple homes?

A: Yes! The number of homes depends on your rank. Use /sethome [name] to set multiple homes.

Q: How do I join tournaments?

A: Check the tournaments section on our website or use /tournament in-game to register.

Q: What happens if I get griefed?

A: Contact staff immediately. We have CoreProtect to rollback any damage to your builds.
